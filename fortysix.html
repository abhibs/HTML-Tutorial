Label

46.1. label tag :-

It defines a set of text that is associated with a particular element.

example :-

<label>â€¦..</label>
Attribute Value Description
for element_id Specifies which form element a label is bound to
form form_id Specifies one or more forms the label belongs to

element_id - The id of the element the label is bound to

coding example :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <label>Male:<input type="radio" name="sex"></label>
        <label>Female:<input type="radio" name="sex"></label>
    </body>
</html>

coding example with for attribute :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <label for="male">Male:</label>
            <input type="radio" name="sex" id="male">
        <label for="female">Female:</label>
            <input type="radio" name="sex" id="female">
    </body>
