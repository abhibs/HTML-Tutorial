Chapter - 19

Blockquote and q

19.1. <blockquote> tag :-

The <blockquote>……</blockquote> is used for long and multiline quotations.

coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <blockquote>Abhiram</blockquote>
    </body>
</html>
output

19.2. <q> tag :-

The <q>…… </q> element is used for shorter quotes that sit within a paragraph.

coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <q>
            Javalli
            Tudoor
            Thirthahalli Tq
            Shimoga D
            577226
        </q>
    </body>
</html>
output
