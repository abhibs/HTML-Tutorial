Chapter - 31

Definition List

31.1. <dl> tag:-

A definition list is a list of terms, with a definition of each term.

We can create definition list by using the <dl>….. </dl> with <dt>  and <dd> tag.

The <dt>….</dt> tag is used to define the term; whereas, the <dd> tag is used to give the term’s definition.

example :-

<dl>
    <dt>Collage</dt>
        <dd>A boring place</li>
    <dt>Library</li>
        <dd>Learn as much as possible</dd>
    <dt>Railway Station</li>
        <dd>too much of croweded</dd>
</dl>

coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <dl>
            <dt>Collage</dt>
                <dd>A boring place</li>
            <dt>Library</li>
                <dd>Learn as much as possible</dd>
            <dt>Railway Station</li>
                <dd>too much of croweded</dd>
        </dl>
    </body>
</html>
