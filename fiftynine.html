Progress Tag

59.1. progress tag :-

The <progress> element represents the completion progress of a task. It has two attributes max which Specifies how much work the task requires in total and value which Specifies how much of the task has been completed. It must be a valid floating point number between 0 and max, or between 0 and 1 if the max attribute is not present. 

example :-

<progress value="50" max="100"></progress>

coding example :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <progress value="75" max="100"></progress>
    </body>
</html>
