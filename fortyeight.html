Legend

48.1. legend tag :-

This tag defines a caption for the fieldset element.

example :-

<fieldset>
    <legend>…….</legend>
</fieldset>

coding example :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <form>
            <fieldset>
                <legend>Personal Information</legend>
                <br>
                    Name: <input type="text" name="fname">
                    Password: <input type="password" name="pass">
                    <br>
            </fieldset>
        </form>
    </body>
</html>
