Datalist Tag

53.1. datalist tag :-

The datalist tag provides an autocomplete feature, which means that it allows you to select an input from the predefined options. The datalist tag is combined with the input tag.

example :-

<datalist>
    <option>………...</option>
    <option>………...</option>
</datalist>

coding example :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <form>
            <input type="text" list="programming">
            <datalist id="programming">
                <option value="Java Programming"></option>
                <option value="C Programming"></option>
                <option value="C++ Programming"></option>
                <option value="C# Programming"></option>
                <option value="HTML"></option>
                <option value="PHP"></option>
            </datalist>
        </form>
    </body>
</html>
