Video tag

39.1. video tag :-

The <video>â€¦. </video> tag is used to play video files.

 Supports: MP4, ogg, WebM.

Attribute Value Description
autoplay autoplay Play a video file immediately or automatically on the loading of  a web page.
controls controls Display the controls on a web page such as a play button etc.
loop loop Specifies that the video will start over again, every time it is finished.
muted muted Specifies that the audio output of the video should be muted.
preload auto
metadata
none
 Specifies if and how the author thinks the video should be loaded when the page loads.
src src
 Specifies the URL of the video file.
width pixels
 Sets the width of the video player.

coding example with controls attribute and controls value :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <h3>Video Files</h3>
        <video controls="controls" src="video.mp4"></video>
    </body>
</html>
