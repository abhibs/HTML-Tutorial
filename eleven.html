Heading

11.1. Heading tag :-

Heading defines the format and structure of a document. 

There are six heading tag: -

  • <h1>…… </h1>
  • <h2>…… </h2>
  • <h3>…… </h3>
  • <h4>…… </h4>
  • <h5>…… </h5>
  • <h6>…… </h6>

Any text inside these tags is displayed differently depending on the heading number.

Note :-

  • <h1>……</h1> is largest heading tag and <h6>……. </h6> is smallest.
  • Heading get their own line on the web pages by starting from a new line.

example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <h1>Abhiram B S</h1>
        <h2>Abhiram B S</h2>
        <h3>Abhiram B S</h3>
        <h4>Abhiram B S</h4>
        <h5>Abhiram B S</h5>
        <h6>Abhiram B S</h6>
    </body>
</html>
