Unordered List

29.1. <ul> tag:-

An unordered list is used to create a bulleted list of items.

The<ul>….. </ul> tag is used to define an unordered list; whereas, the <li>…</li> tag is used to define the items of the list.

example :-

<ul>
    <li>Linux</li>
    <li>Windows</li>
    <li>Mac</li>
    <li>Android</li>
</ul>

coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <ul>
            <li>Linux</li>
            <li>Windows</li>
            <li>Mac</li>
            <li>Android</li>
        </ul>
    </body>
</html>
output
