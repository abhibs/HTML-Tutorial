Fieldset

47.1. fieldset tag :-

This creates a box around the related form controls.

example :-

<fieldset>……</fieldset>
Attribute Value Description
disabled disabled Specifies that a group of related form elements should be disabled
form form_id Specifies forms the fieldset belongs to
name text Specifies a name for the fieldset

coding example :-


<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <form>
            <fieldset>
                Name: <input type="text" name="fname">
                <br>
                Password: <input type="password" name="pass">
                <br>
                City: <input type="text" name="city"><br>
            </fieldset>
        </form>
    </body>
</html>
output
