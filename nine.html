Chapter - 9

HTML 5

9.1. HTML 5 :-

HTML 5 enables to create more interactive websites by embedding audio, video and graphics on the web page.

The purpose of HTML5 is primarily to make it easier for web developers.

New features:

  • Browser Support
  • Geo location
  • Offline Application Cache
  • Web Storage
  • Error Handling
  • Ne Application Programming Interface (API)
  • New Structure

9.2. Application Programming Interface (API) :-

  • 2D drawing on a web page
  • Drag and Drop
  • Timed Media Playback
  • Browser History Management

9.3. New Structure :-

  • <article>
  • <aside>
  • <header>
  • <details>
  • <dialog>

9.4. Example :-

<!DOCTYPE html>
<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        <h1> Body of Page </h1>
    </body>
</html>

9.5. DOCTYPE declaration :-

<!DOCTYPE html>

It is an instruction to the web browser about what version of HTML the page is written in.

The <!DOCTYPE> declaration is not case sensitive.
