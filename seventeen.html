Italic

17.1. <i> tag :-

The <i>…… </i> tag is used to render the text as italic.

coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        My name is : <i>Abhiram B S</i>
    </body>
</html>
output

combination of <b> and <i> tag coding example :-

<html>
    <head>
        <title>Abhiram</title>
    </head>
    <body>
        My name is : <b><i>Abhiram B S</i></b>
    </body>
</html>
output
